# install-docker
install-docker是一个基于docker官网的安装脚本，主要是方便docker通过这个脚本一键安装。
这个脚本只适用于Debian11以上的系统，其他Linux开发版不支持。

食用前，需要授权脚本权限
sudo chmod +x install-docker.sh
然后运行脚本
sh install-docker.sh
